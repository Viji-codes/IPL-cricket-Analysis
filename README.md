# 🏏 IPL Cricket Analytics (2008-2013)

## 📌 About This Project
This project analyzes IPL Cricket statistics 
from 2008 to 2013 using real match data to 
identify top performers of the early IPL era!

## 🎯 What This Project Shows
- 🏏 Top 10 Run Scorers (2008-2013)
- 🎯 Top 10 Wicket Takers (2008-2013)
- 💰 Top 10 Most Expensive Players

## 🛠️ Tools & Technologies Used
- Python 🐍
- Pandas
- Matplotlib

## 📊 Charts Generated
- top_batsmen.png — Top Run Scorers
- top_bowlers.png — Top Wicket Takers
- top_expensive.png — Most Expensive Players

## 💡 Interesting Findings
- Tendulkar SR was the highest run scorer!
- Early IPL dominated by international legends!
- Dataset covers IPL seasons 2008 to 2013
